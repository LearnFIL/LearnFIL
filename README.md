# LearnFIL

LearnFIL is a learning platform for understanding and building with the Filecoin ecosystem.

It combines structured lessons, interactive exercises, and an AI learning companion to help developers move from concepts → real applications.

![LearnFIL](https://img.shields.io/badge/Filecoin-Education-orange)
![License](https://img.shields.io/badge/license-MIT-blue)

## Ecosystem

## Core Features

* **Structured Learning Modules**
  Guided lessons covering Filecoin concepts and development workflows.

* **Interactive Code Exercises**
  Hands-on examples that allow learners to experiment directly in the browser.

* **Progress Tracking**
  Track lesson completion and learning milestones.

* **Learny AI Companion**
  Ask questions about Filecoin concepts and receive educational explanations.

## Local Development

Clone the repository:

```bash
git clone https://github.com/LearnFIL/LearnFIL.git
cd LearnFIL
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
