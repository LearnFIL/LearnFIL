# Development

This guide explains how to run LearnFIL locally for development.

---

## Live Platform

LearnFIL is available online:

[https://learn-fil.xyz/](https://learn-fil.xyz/)

You can explore the platform without running it locally.

---

## Prerequisites

Install the following tools:

* Node.js (v18 or later)
* npm

---

## Clone the Repository

```bash
git clone https://github.com/LearnFIL/LearnFIL.git
cd LearnFIL
```

---

## Install Dependencies

```bash
npm install
```

---

## Start the Development Server

```bash
npm run dev
```

The application will start locally and can be accessed in your browser.
