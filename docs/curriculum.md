# Curriculum

LearnFIL provides structured learning modules designed to help developers understand and build with the Filecoin ecosystem.

The curriculum progresses from foundational concepts to practical application development.

---

## Module 1 — Content Identifiers (CIDs)

Introduces the concept of content addressing and how data is identified in decentralized systems.

Topics include:

- What a CID is
- Why content addressing matters
- CID structure
- Multihash and multicodec basics
- Creating and validating CIDs

---

## Module 2 — Storage on Filecoin

Explores how data storage works in the Filecoin network.

Topics include:

- Storage providers
- Storage deals
- Data persistence
- Filecoin storage lifecycle

---

## Module 3 — Filecoin Virtual Machine (FVM)

Introduces programmable capabilities within the Filecoin ecosystem.

Topics include:

- FVM overview
- Smart contracts on Filecoin
- Interaction with built-in actors
- Building storage-enabled applications

---

## Module 4 — IPFS and Filecoin

Explains how IPFS and Filecoin complement each other.

Topics include:

- IPFS fundamentals
- Content routing
- Persistence strategies
- Integrating IPFS with Filecoin storage

---

## Module 5 — Data Retrieval

Focuses on accessing and retrieving stored data efficiently.

Topics include:

- Retrieval mechanisms
- Data availability
- Retrieval providers
- Performance considerations

---

## Module 6 — Storage Proofs

Introduces the cryptographic mechanisms that make Filecoin verifiable.

Topics include:

- Proof of Replication
- Proof of Spacetime
- Sealing process
- Verification concepts

---

## Module 7 — Building Applications

Focuses on building real applications that use Filecoin storage.

Topics include:

- Application architecture
- Storage integration
- Deployment strategies
- Best practices for production systems

---

## Future Modules

Planned areas of expansion include:

- Advanced FVM development
- Filecoin economics
- Storage provider operations
