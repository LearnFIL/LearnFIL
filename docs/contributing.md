# Contributing

Thank you for your interest in contributing to LearnFIL.

Contributions help improve the learning experience and expand the platform for the Filecoin developer community.

---

## Ways to Contribute

You can contribute in several ways:

- Improve lesson content
- Add new learning modules
- Fix bugs or improve the interface
- Improve documentation
- Suggest new features

---

## Getting Started

1. Fork the repository
2. Create a new branch for your change
3. Make your changes
4. Submit a pull request

Example workflow:

```

git checkout -b feature/my-improvement
git commit -m "feat: improve lesson explanation"
git push origin feature/my-improvement

```

---

## Contribution Guidelines

Please follow these principles when contributing:

- Keep explanations clear and beginner-friendly
- Prefer small, focused pull requests
- Write descriptive commit messages
- Update documentation when needed

---

## Reporting Issues

If you find a bug or have a suggestion:

1. Open a GitHub issue
2. Describe the problem clearly
3. Include steps to reproduce if applicable

---

## Community

LearnFIL is an open project focused on helping developers learn the Filecoin ecosystem.

Constructive feedback and collaboration are always welcome.
